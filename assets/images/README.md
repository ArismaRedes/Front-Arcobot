# Imagenes UI de ArcoBot

Pon tus imagenes en estas rutas exactas para que la app las use automaticamente:

- `assets/images/auth/student_login_hero.png`
  - Uso: portada del login de estudiantes.
  - Sugerido: 1600x1000 px, estilo horizontal.

- `assets/images/auth/teacher_login_hero.png`
  - Uso: portada del login de profesores.
  - Sugerido: 1600x1000 px, estilo horizontal.

- `assets/images/dashboard/dashboard_hero.png`
  - Uso: banner principal del dashboard.
  - Sugerido: 1600x900 px.

- `assets/images/brand/mascot.png`
  - Uso: mascota/branding (reservado para componentes futuros).
  - Sugerido: PNG transparente 800x800 px.

- `assets/images/brand/arcobot_animado.svg`
  - Uso: ilustracion principal de la pantalla de inicio (actualmente en uso).
  - Sugerido: viewBox cuadrado y pesos bajos para mejor rendimiento.

Formatos recomendados:
- PNG para ilustraciones con transparencia.
- JPG para fotos pesadas.
- WEBP si quieres reducir peso.

Buenas practicas:
- Mantener cada imagen por debajo de 500 KB cuando sea posible.
- Evitar texto incrustado en la imagen para mejor accesibilidad.
